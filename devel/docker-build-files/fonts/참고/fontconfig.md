# fontconfig

## 설치

`$ yum install fontconfig`

## 종속 패키지

`$ yum install -y dejavu-fonts-common dejavu-sans-fonts fontpackages-filesystem freetype libpng`

## fontconfig 설치 후

```bash
$ fc-list
```
